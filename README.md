# Webová To‑Do aplikace (Python + Flask)

Jednoduchá webová aplikace v Pythonu s frameworkem Flask.  
Na straně serveru generuje HTML stránku se seznamem úkolů.

## Funkce
- zobrazení seznamu úkolů
- přidání úkolu
- smazání úkolu

## Spuštění

1. Nainstaluj Flask:
   pip install flask

2. Spusť aplikaci:
   python app.py

3. Otevři v prohlížeči:
   http://127.0.0.1:5000
